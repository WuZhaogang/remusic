# remusic
- 增加网络浏览歌单，专辑，多任务下载歌曲，曲库搜索歌曲等，使用百度api，（网易云api参考现有的破解，AES可以加密成功，RSA使用java的工具，加密后不对，使用bigdemical纯数学计算，加密成功，但在手机上加密时间太久，有谁知道怎么在安卓上怎么用，请告知，谢谢）

- 原来本地播放器版本保存在remusic_local
